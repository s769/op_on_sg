recursions.mem2(func)[source]¶
This is a decorator function that memoizes recursive functions that

return arrays of values from index 0 to j. If the recursion has already been called on a higher index, this decorator will cause the function to automatically return those precomputed values.

Example: Suppose mem2 decorates the function alpha_array(j),

which returns [alpha_0, … , alpha_j]. Calling alpha_array(10) will return [alpha_0, … , alpha_10]. Calling alpha_array(5) immediately after will return [alpha_0, … , alpha_10] without having to compute those values again.

monomials.generate_T(level, deg, frac=True)[source]¶
This function calculates the values of the monomials up to a certain

degree at a given level of SG.

Parameters

  • level – level of SG required

  • deg – maximum degree monomial required

  • frac – Boolean representing whether the coefficients should remain as fractions or should be converted to floating point numbers at the end of all calculations.

Returns

np.array with dimensions 3 x 3^(level+1) x deg+1.

The kth page of T has values of the monomials P_jk (j = 0…deg) at each of the 3^(level+1) TLR indices of points on the given level of SG.

Return type

T
